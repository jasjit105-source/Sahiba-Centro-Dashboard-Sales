# Sahiba Centro Dashboard

Store operations dashboard for SAHIBA wholesale fashion (Mexico City).
This is the owner/manager version with full access to all data including costs and profits.

## Features
- Sales tracking across all stores
- Inventory management
- Corte de Caja (cash register reconciliation)
- Store performance comparison
- Commission tracking (agents: Jazmin, Yoana, Nancy)
- Style analysis

## Tech Stack
- Single-file React app (Babel in-browser)
- Netlify Functions (API proxy to SQL Server via ngrok)

## Stores
- Leona Vicario (TS0001)
- Circunvalación (TS0002)
- Lecumberri (TS0003)
- Chinconcuac/Izazaga (TS0004)

## Deployment
Deploy to Netlify by uploading the zip or connecting this repo.
